# Ray哥分析模式拆解

> 抽样范围：2024-04 / 2024-08 / 2024-11 / 2024-12 / 2025-04 / 2025-09 / 2025-12 / 2026-03 / 2026-04 / 2026-05 月度文摘 + 7 个 archetype 样本 + 跨月度的 grep（FOMC、CPI、NFP、关税、回购）。视频帖（706 条 / 28%）只有标题没有正文，读不到内容，因此模式提取偏向文字帖，可能低估 Ray 在视频里的"短线择时"比例。

每个模式格式：
- **输入**：触发它的数据/事件
- **输出形态**：Ray 给出的结论形态
- **复用度**：A=每次都套用 / B=经常但要看条件 / C=一次性叙事
- **skill 候选**：是 / 否 / 边缘 + 简短理由

---

## A. 框架级（reusable，最有价值）

### A1. 大非农"企业 vs 家庭调查"二元拆解
- **输入**：BLS NFP 报告（每月第一个周五）
- **输出形态**：
  - 用 establishment（按岗位、含非法移民/兼职）vs household（按人头、不含非法移民）的差异解释强弱
  - 对一次性冲击（飓风、波音罢工、政府停摆）→ "establishment 受打击、household 不受影响" → 失业率往往原地不动
  - 修订方向预判：选举后 / 政府换届时倾向于"开始下修"
- **示例**：2024-04-07（3 月 NFP 30.3 万）、2024-11-01（10 月 12K 飓风+罢工）、2024-12-08（11 月 ADP）、2025-09-05（8 月 22K 触发 Sahm）
- **复用度**：A
- **skill 候选**：**是**。结构化非常强，每次出来都能套，且 Claude 直接套用比临时推导稳。

### A2. CPI/PPI"核心 vs 名义、商品 vs 服务"分项框架
- **输入**：CPI/PPI 月度报告
- **输出形态**：
  - 提前发布前给出市场一致预期 + 投行分歧（BAML 偏低 / Bloomberg 偏高 / Cleveland Fed Nowcast）
  - 关注 SuperCore（剔除房租的核心服务）作为 Powell 的真心数据
  - PPI 拆分：服务 vs 商品；商品里看汽油 vs 食品；如果汽油 PPI 与零售汽油价格背离 → 怀疑季调黑盒
  - "核心 CPI 0.3% 是 Fed 容忍线，0.4% 就要重定价降息路径"
- **示例**：2024-04-09 / 2024-04-10 / 2025-09-11
- **复用度**：A
- **skill 候选**：**是**。市场反应阈值（0.3% / 0.4%）和分项导读模板高度可复用。

### A3. FOMC "Skip → Hold → Pause" 措辞解码
- **输入**：FOMC 决议 + Powell 记者会
- **输出形态**：从鸽到鹰排序：Skip < Hold < Pause；用记者会措辞预判后续 1-2 次会议方向
- **示例**：2024-12-19（"Pause"=明年降息次数从 4 降到 2）、2025-09-17（risk management 框架）、2025-12-10（RMP 抢戏）
- **复用度**：A
- **skill 候选**：**是**。措辞 → 路径预期映射很值得固化。

### A4. 国债拍卖五维结构解读
- **输入**：每周 / 每月美国国债（2Y / 5Y / 7Y / 10Y / 20Y / 30Y）拍卖结果
- **输出形态**：固定五维：
  1. High yield vs WI yield → tail（正负基点 + 历史排序）
  2. Bid-to-cover ratio vs 6 次平均
  3. Indirect bidder（外国央行）份额 vs 趋势
  4. Direct bidder 份额
  5. Dealer take（high = 滞销坏信号）
  - 然后映射到收益率方向 + 国外需求结构判断
- **示例**：贯穿 2024-04 到 2026-05，几乎每次拍卖都套
- **复用度**：A
- **skill 候选**：**是**。这是最机械、最稳定、价值密度最高的 Ray 模板，Claude 自己写非常容易写错或漏维度。

### A5. 流动性结构链：RRP → 银行准备金 → 短债发行 → 股市
- **输入**：财政部融资计划、RRP 余额、SRF/IORB、SOFR
- **输出形态**：
  - 发短债 vs 发长债的二分法（耶伦偏短、贝森特偏长）
  - "短债覆盖赤字 → 推 RRP 释放流动性 → 利好股市；长债覆盖 >100% 赤字 → 流动性紧 → 股市承压"
  - 引 Clarida、Cabana 的研究
  - 把 RMP / Term Repo / IORB 调整都视为隐性 QE
- **示例**：2024-12-31 流动性预警、2025-09-04 短债负面影响、2025-12-09 / 2025-12-11 RMP 提前预言
- **复用度**：A
- **skill 候选**：**是（最强候选）**。这是 Ray 最有"信息差"感觉的领域，且 Claude 自己拼这条链需要多源证据。

### A6. 数据预判 → 事后复盘"三段式"
- **输入**：任一即将发布的高频数据
- **输出形态**：
  - 发布前帖：列各家投行预期、Cleveland Fed Nowcast、提示"哪个分项是关键"
  - 发布当天帖：实际数据 + 是否超预期 + 与 Fed 路径关系
  - 1-2 天后帖：市场反应解读，期货持仓变化
- **示例**：每月 CPI / NFP / ADP / 零售 / PCE 都是这个三段
- **复用度**：A
- **skill 候选**：**边缘**。三段式本身有用，但每次都涉及具体数据，更适合作为 A2/A1 的 wrapper 而不是独立 skill。

### A7. 硬数据 vs 软数据背离框架
- **输入**：PMI（标普 vs ISM）、消费者信心、调查类 vs 实际订单/支出
- **输出形态**："软的不行，硬的还行" 或反之 → 用来解释短期波动 vs 趋势分歧
- **示例**：2025-04-01 PMI、2025-09-02 8 月 PMI、2026-05 多次
- **复用度**：A
- **skill 候选**：**否**。框架很简单，Claude 自己就能套，固化成 skill 没有边际收益。

### A8. VIX 对冲配置法
- **输入**：股票多头组合 + 当前 VIX 水平
- **输出形态**：低 VIX 时配 3-5% OTM CALL，亏完就这么多；股债双杀时变成 15-25% 回报对冲股票多头
- **示例**：2024-04-11（10Y 拍卖暴雷当天 VIX cover loss）、2024-04-16、贯穿全期
- **复用度**：B
- **skill 候选**：**否**。这是 portfolio rule，不是市场分析。值得记成一条 user 偏好。

## B. 重复主题/叙事（B 级，复用但更靠观点）

### B1. "拜登劳工局/政府数据造假"长期主题
- **输入**：任一官方就业 / 通胀数据
- **输出形态**：默认怀疑数据被人为做高 → 等 BLS 年度修订下修来"打脸"
- **里程碑**：2024-08 BLS 修正 -818K（费城联储提前预言）、2025-09-09 又一次修订
- **复用度**：B
- **skill 候选**：**否**。带政治色彩，且本质是单一假设；做成 skill 会固化偏见。可以作为"BLS 修订日历"提醒，不是 skill。

### B2. 历史类比模板
- **输入**：当前事件
- **输出形态**：找一个历史类比（1929 Smoot-Hawley、70 年代双峰 CPI、2020 苏莱曼尼空袭、2008/2010 适度宽松）
- **复用度**：B
- **skill 候选**：**否**。Claude 本能就在做类比，且 Ray 的类比常常是叙事性而非严格的 base rate。

### B3. 中国宏观刺激日历
- **输入**：政治局会议 / 中央经济工作会议 / 两会 / NPC 常委会
- **输出形态**：会议时间表 + 宽松基调升级语言（"谨慎"→"适度宽松"→"更加积极"）+ 化债额度 vs 新增刺激区分
- **示例**：2024-11-08 NPC 10 万亿、2024-12-09 政治局"适度宽松"
- **复用度**：B
- **skill 候选**：**边缘**。日历型 skill 有点价值，但 Ray 的解读偏定性。可以做成"中国财政货币会议日历 + 关键词分级"。

### B4. 央行购金 + 黄金交易场景
- **输入**：每月央行购金数据、SGE 离岸金库进展、印度/巴西减持美债
- **输出形态**：把 CB diversification → ETF flow → 投机仓位 拆成三股推力
- **复用度**：B
- **skill 候选**：**否**。每次都引高盛/渣打目标价，本身没有 Ray-specific edge。

### B5. AI/半导体主线追踪
- **输入**：NVDA 财报、token 调用量、Korea KOSPI、HBM 厂商动态
- **输出形态**：从"NVDA = AI"逐步迁移到"半导体 = AI = 全球指数核心"，2026Q1 转向"NVDA 出问题，韩股 / 中国推理模型崛起"
- **复用度**：B
- **skill 候选**：**否**。主题驱动型分析，不能模板化。

## C. 一次性 / 叙事型（C 级，不值得 skill 化）

- **C1. 美国大选预测**：超自然 + 战略分析家 Jim Rickards 等小道消息（2024-11-03）
- **C2. 巴菲特 13F 解读**："卖出美国"（2024-11-04）
- **C3. 牧师 Brandon、Jim Rickards 类玄学预测**：2024-11-03 多次
- **C4. 美国回购供需拐点**（2026-04-26）：一次性框架性发文
- **C5. 关税违背凯恩斯+美元霸权理论**（2025-04-06）：单点经济学论证
- **C6. 算力 ≠ 收入 反驳黄仁勋**（2026-03-05）：单点反驳
- **C7. "解放日"复盘**（2025-04-05）：四大紧缩同时发酵 → 短线发财

这些都是高密度的 one-off 帖，质量参差，但作为"Claude skill"没有可复用骨架——属于 Ray 的输出，不属于 Ray 的方法。

## 跨模式观察

1. **Ray 的"信息源画像"高度重复**：
   - 美联储传声筒 Nick Timiraos（WSJ）
   - 美银 Michael Hartnett 的周度 Flow Show（每周必引）
   - 高盛交易台备忘录
   - BAML 零售预测
   - 摩根斯坦利衰退概率
   - 渣打银行 Steve Englander
   - BoA Mark Cabana（利率策略）
   - ZeroHedge 风格的拍卖结果数据
   - 这些是他的"工具箱"——很多 skill 设计应当让 Claude 也去 fetch 这些固定来源。

2. **Ray 的视角偏向**：
   - 反拜登 / 反民主党、亲懂王（贯穿 2024-2025）
   - 看多黄金、看多核能、看多半导体（持续 25 个月）
   - 中长期"美国经济末期"叙事，但短线常常被打脸（很多次"美股要回调"call 紧接着是新高）
   - 这是 alpha source 还是 bias 取决于读者怎么用，但做 skill 时不应内化这些立场

3. **survivorship bias 提醒**：
   - 圈子帖是 Ray *愿意发*的内容
   - 看不到他的删帖、未发的"打脸"call、内部群里的失败操作
   - engagement 普遍是个位数 → 这是定价 / VIP 信号源而非"专家共识"，不能用点赞数判断质量

4. **方法 vs 信息**：
   - 大约 60% 的内容是"信息整合"（投行报告、宏观数据、新闻）
   - 30% 是"框架应用"（A1-A8）
   - 10% 是"独立观点"（如关税违背美元霸权、算力≠收入）
   - 真正能 skill 化的是中间 30%；前 60% Claude 用 web search + fetch 就能复制；后 10% 是 Ray 自己的洞察，不可模板化
